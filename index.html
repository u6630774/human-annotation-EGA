Online Human Annotation Pages

How to use

  1. Upload these HTML files and your image folder to any static hosting service (GitHub Pages, Netlify, Vercel, lab server, etc.).
  2. Deploy the included google_apps_script.gs as a Google Apps Script Web App connected to a Google Sheet.
  3. Paste the image base URL and Apps Script URL below, then click “Generate annotator links”.
  4. Send each annotator their personal link. They can submit directly to Google Sheet and still export a CSV backup.
Example image base URL: https://your-site.com/images
Example Apps Script URL: https://script.google.com/macros/s/XXXXXXXX/exec
